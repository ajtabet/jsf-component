jsf-component
---------------------------------------
http://code.google.com/p/jsf-component/

Custom JSF component - multiple-selection component that consists of two lists and add/remove buttons. One list contains available (not selected) options. Another list contains selected options. Options can be moved from one list to another by clicking on the add/remove buttons.

This project is developed in Red Hat Developer Studio.
